# מטרת הפרויקט — Abandoned Checkout Worker

## רקע
המערכת מטפלת בנטישות עגלה מחנויות Shopify.
הודעות הנטישה מגיעות דרך Google Pub/Sub לוורקר ב-Cloudflare,
שמחליט האם להעביר אותן ל-Webhook של מערכת הטיפול בלקוחות.

## הבעיה שנפתרה
לא כל חנות צריכה לקבל טיפול בנטישות:
- חנויות עם סטטוס **uninstall** — הסירו את האפליקציה
- חנויות עם סטטוס **silent-off** — ביקשו להשתיק

בנוסף, אין טעם לטפל בנטישה שאין לה מספר טלפון.

## הפתרון
Cloudflare Worker שיושב בין Pub/Sub ל-Webhook ומסנן:
1. קורא את סטטוס החנות מ-Google Sheets
2. חוסם חנויות עם סטטוס uninstall / silent-off
3. חוסם הודעות ללא מספר טלפון
4. מעביר רק מה שצריך טיפול

## תוצאה
הפחתת עומס מיותר על מערכת הטיפול בלקוחות,
ומניעת פנייה ללקוחות של חנויות שלא הסכימו לכך.
